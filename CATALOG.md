## HOW TO USE THIS CATALOG

1. **Find a skill** — search by category above or keyword in the `skills/` folder
2. **Find an agent** — pick by role from the Development, Operations, or Security groups
3. **Build a loadout** — see [LOADOUTS.md](LOADOUTS.md) for pre-built bundles
4. **Install** — copy `agents/` to `~/.claude/agents/` and `skills/` to `~/.claude/skills/`
