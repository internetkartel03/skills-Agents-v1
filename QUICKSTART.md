# Zouk Skills & Agents — Quick Start

## Step 1

Review:

- README.md
- CORE.md
- LOADOUTS.md
- CATALOG.md

These files explain how the system is organized.

## Step 2

Choose a workflow style.

Examples:

- Open WebUI
- Claude Code
- Codex
- Local AI environment
- Internal team workflow

## Step 3

Start with a small loadout.

Avoid loading the entire library at once.

Recommended starting point:

- workspace-surface-audit
- skill-scout
- agent-sort
- prompt-optimizer
- plan-orchestrate

## Step 4

Add project-specific skills and agents only when needed.

Keep the operating layer small and focused.

## Step 5

Review hooks, scripts, and automation components before enabling them.

Do not execute unknown automation blindly.

## Recommended Progression

Beginner:
- Core layer only.

Intermediate:
- Core + project loadouts.

Advanced:
- Custom routing, custom loadouts, and custom agent teams.

## Goal

The objective is not to use every skill.

The objective is to create an efficient workflow using the smallest useful set of tools.
